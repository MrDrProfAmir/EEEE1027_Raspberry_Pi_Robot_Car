This repository contains the Raspberry Pi source code developed for the EEEE1027 module project.

Each file corresponds to a specific task:

PWM_Angle.py - PWM-based motor control and angle rotation
Line_Following.py - basic line-following implementation
Symbol_Shape_Detection_Combined.py - symbol and shape detection using computer vision
Line_Following_Symbol_Detection_Combined.py - integrated system for coloured line following and symbol detection
